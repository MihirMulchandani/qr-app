QR is a simple browser tool that turns text or links into QR codes.

No accounts.  
No ads.  
No subscriptions.

Just enter your text and generate a QR instantly.

Everything runs directly in the browser.  
Nothing is stored or sent to a server.

### What it does

• Convert text into a QR code  
• Convert links into a QR code  
• Generate instantly in the browser
